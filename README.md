# Harry Potter World

**Функционал проекта**: <br>
- Отображение персонажей<br>
- Отображение подробной информации о персонаже<br>
- Поиск персонажей <br>
- Добавление персонажей в избранное<br>
- Выбор цветовой темы<br>

**Используемые технологии**: 
- HTML<br>
- CSS<br>
- React<br>
- React-Router<br>
- Redux<br>
- Alias<br>
